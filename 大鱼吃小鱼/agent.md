# 大鱼吃小鱼 项目说明

## 项目简介
一个基于 HTML5 Canvas 的休闲游戏。玩家控制主鱼，通过鼠标移动吞噬等级不超过当前主鱼等级的敌鱼。目标是让主鱼成长到最高等级并吃掉最大的敌鱼，触发恋爱主题胜利特效。

## 主要文件
- `index.html` - 游戏页面结构
- `style.css` - 游戏界面样式与 HUD 布局
- `script.js` - 游戏逻辑、渲染、碰撞、粒子特效
- `1.png`、`2.png`、`3.png`、`4.png`、`主鱼.png` - 游戏图像资源

## 核心玩法
- 主鱼随着吃掉敌鱼逐渐成长
- 当前主鱼等级决定可吞噬的鱼级别
- 吃掉与当前等级相同的鱼会升级主鱼等级
- 吃掉最大等级鱼并且主鱼达到最高等级时胜利
- 胜利时显示浪漫特效：心形粒子飞散

## 设计要点
- `script.js` 中 `PLAYER_GROWTH` 调整为 `0.55`，让成长节奏更慢
- `SPAWN_INTERVAL` 和 `MAX_ENEMIES` 控制敌鱼生成节奏
- `gameLoop` 使用 `requestAnimationFrame` 保持流畅动画
- 胜利后显示 overlay 并停止动画
